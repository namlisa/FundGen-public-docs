# צפייה בהתקדמות התרומות שלך

## **גישה להתקדמות**

1.  עבור ללשונית **תרומות**
2.  לחץ על לשונית **היעד שלי**

## **הסבר המדדים**

| **מדד** | **משמעות** |
| :------ | :--------- |
| **גויס** | סכום כולל שגייסת |
| **יעד** | היעד האישי שלי |
| **% התקדמות** | אחוז לעבר היעד |
| **נשאר** | סכום שנותר |
| **תורמים** | מספר אנשים שתרמו |
| **ממוצע** | גודל תרומה ממוצע |

## **מכפילים**

אם לקמפיין יש התאמת תרומות:

-   מכפיל 2x אומר ש-1,000$ שלך נספרים כ-2,000$
-   מוצג: "גויס: 2,000$ (כולל התאמה 2x)"

## **רענון נתונים**

-   נתונים מתעדכנים אוטומטית
-   משוך למטה לרענון ידני
-   סנכרון שעתי עם הפלטפורמה לדיוק

## **הגדרת היעד שלי**

היעד שלי נקבע על ידי:

-   רישום בפלטפורמה (מיובא אוטומטית)
-   התאמת מנהל הקמפיין

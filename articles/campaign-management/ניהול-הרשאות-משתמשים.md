# User Role Management

## User Roles

The FundGen Ambassador App supports different user roles, each with specific access levels to campaign features and data.

| **Role** | **Access Level** |
|---|
| Admin | Full control over campaign settings, ambassador management, data access, and user roles. |
| Team Leader | Manage ambassadors, assign contacts, view reports, and access campaign data. |
| Ambassador | Access personal campaign performance, assigned contacts, and make donations. |

## Permission Areas

Permissions are categorized into the following areas:

### Campaign Settings
*   Configure general campaign settings, manage integrations, set notifications, define goals and dates, and **assign WhatsApp bot message senders**.

### Ambassador Management
*   Invite ambassadors
*   Remove ambassadors
*   Assign contacts
*   View all ambassador activity

### Data Access
*   View all donations
*   Export data
*   View contact details
*   Access reports

### Deep Dive Search Access
*   Control whether ambassadors can use the AI-powered 'Deep Dive Search' feature on contacts.

### Payment Processing (Tap to Pay)
*   Allow or restrict ambassadors from processing donations using the 'Tap to Pay' feature.

## Changing User Roles and Custom Permissions

You can customize user roles and permissions, including granular access to features like **Deep Dive Search**, **Tap to Pay**, and **WhatsApp bot sender assignments**, from the FundGen admin interface.

1.  Go to the **Admin interface** > **Ambassador Management**.
2.  Click on the user whose **Management Permissions** you want to edit.
3.  Select the desired permissions.
4.  Save your changes.

# ניהול הרשאות משתמשים

## תפקידי משתמש

אפליקציית FundGen Ambassador תומכת בתפקידי משתמש שונים, שלכל אחד מהם רמות גישה ספציפיות לתכונות ונתוני הקמפיין.

| **תפקיד** | **רמת גישה** |
|---|
| מנהל מערכת | שליטה מלאה על הגדרות הקמפיין, ניהול שגרירים, גישה לנתונים ותפקידי משתמשים. |
| מנהל צוות | ניהול שגרירים, הקצאת אנשי קשר, צפייה בדוחות וגישה לנתוני קמפיין. |
| שגריר | גישה לביצועי הקמפיין האישיים, אנשי קשר שהוקצו לו וביצוע תרומות. |

## תחומי הרשאות

ההרשאות מחולקות לתחומים הבאים:

### הגדרות קמפיין
*   הגדרת פרטי קמפיין כלליים, ניהול אינטגרציות, הגדרת התראות, קביעת יעדים ותאריכים, ו**הקצאת שולחי הודעות לבוט וואטסאפ**.

### ניהול שגרירים
*   הזמנת שגרירים
*   הסרת שגרירים
*   הקצאת אנשי קשר
*   צפייה בכל פעילות השגרירים

### גישה לנתונים
*   צפייה בכל התרומות
*   ייצוא נתונים
*   ראיית פרטי אנשי קשר
*   גישה לדוחות

### גישת חיפוש מעמיק (Deep Dive Search)
*   שליטה האם שגרירים יכולים להשתמש בתכונת 'חיפוש מעמיק' מבוססת AI על אנשי קשר.

### עיבוד תשלומים (Tap to Pay)
*   אפשר או הגבל שגרירים מלעבד תרומות באמצעות תכונת 'הקש לשלם'.

## שינוי תפקידי משתמש והרשאות מותאמות

באפשרותך להתאים תפקידי משתמש והרשאות, כולל גישה פרטנית לתכונות כמו **חיפוש מעמיק**, **הקש לשלם**, ו**הקצאות שולחי בוט וואטסאפ**, מממשק הניהול של FundGen.

1.  עבור ל**ממשק ניהול** > **ניהול שגרירים**.
2.  לחץ על המשתמש שאת **הרשאות הניהול** שלו ברצונך לערוך.
3.  בחר את ההרשאות הרצויות.
4.  שמור את השינויים.

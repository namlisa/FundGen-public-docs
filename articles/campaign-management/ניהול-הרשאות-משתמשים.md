# User Role Management

FundGen now offers a flexible and granular permission system, allowing campaign administrators to define custom roles and assign precise access levels to different parts of the platform. This enhances security and workflow efficiency.

## User Roles

App FundGen Ambassador supports different user roles, now including highly customizable roles defined by campaign administrators.

| **Role Type**     | **Description**                                                                                                                                                                                                                                           |
| :---------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Default Roles** | These are the foundational roles (Admin, Team Leader, Ambassador) that provide a base level of access. They are still functional but can be overridden or supplemented by custom roles.                                                                        |
| **Custom Roles**  | Defined by campaign administrators, these roles offer granular control with specific `read`, `write`, or `no access` permissions across various modules. Custom roles are ideal for tailoring access to specific job functions within your campaign team. |

## Understanding Permissions

Permissions are now categorized into specific areas within the FundGen admin interface, and for each module, you can define `read`, `write`, or `no access` privileges.

| **Permission Area**       | **Description**                                                | **Granular Access Options** |
| :------------------------ | :------------------------------------------------------------- | :-------------------------- |
| **Campaign Settings**     | Edit campaign details, manage integrations, set notifications. | Read, Write, No Access      |
| **Ambassador Management** | Invite, remove, assign contacts, view ambassador activity.     | Read, Write, No Access      |
| **Data Access & Reports** | View donations, export data, access contact details, reports.  | Read, Write, No Access      |
| **Communication Tools**   | Manage message templates, call scripts, scheduled messages.    | Read, Write, No Access      |
| **User Role Management**  | Create and assign custom roles, manage user permissions.       | Read, Write, No Access      |

## Creating and Assigning Custom Roles

The previous binary toggles for 'Admin' and 'Team Leader' permissions have been replaced by a comprehensive permission matrix within the new Custom Roles management interface.

To create new custom roles and assign them to users:

1.  Go to the **Admin interface**.
2.  Navigate to **Settings** > **Custom Roles** to define new roles and set their granular permissions. For detailed steps, see [Creating and Managing Custom Roles](/articles/campaign-management/creating-and-managing-custom-roles).
3.  Once roles are defined, go to **Contacts** > **Manage Ambassadors**.
4.  Click on the user whose permissions you want to edit.
5.  Assign the desired custom roles from the available options. For detailed steps, see [Assigning Custom Roles to Users](/articles/campaign-management/assigning-custom-roles-to-users).
6.  Save your changes.

# ניהול הרשאות משתמשים

FundGen מציעה כעת מערכת הרשאות גמישה ומפורטת, המאפשרת למנהלי קמפיין להגדיר תפקידים מותאמים אישית ולהקצות רמות גישה מדויקות לחלקים שונים של הפלטפורמה. זה משפר את האבטחה והיעילות התפעולית.

## תפקידי משתמש

אפליקציית FundGen Ambassador תומכת בתפקידי משתמש שונים, כולל כעת תפקידים הניתנים להתאמה אישית על ידי מנהלי קמפיין.

| **סוג תפקיד**          | **תיאור**                                                                                                                                                                                                                                             |
| :---------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **תפקידי ברירת מחדל** | אלו הם תפקידי הבסיס (מנהל מערכת, מנהל צוות, שגריר) המספקים רמת גישה בסיסית. הם עדיין פונקציונליים אך ניתנים לשינוי או השלמה על ידי תפקידים מותאמים אישית.                                                                                                |
| **תפקידים מותאמים אישית** | מוגדרים על ידי מנהלי קמפיין, תפקידים אלו מציעים שליטה מדויקת עם הרשאות ספציפיות של `קריאה`, `כתיבה`, או `ללא גישה` על פני מודולים שונים. תפקידים מותאמים אישית אידיאליים להתאמת הגישה לתפקידים ספציפיים בצוות הקמפיין שלך. |

## הבנת הרשאות

ההרשאות מחולקות כעת לתחומים ספציפיים בתוך ממשק הניהול של FundGen, ועבור כל מודול ניתן להגדיר הרשאות `קריאה`, `כתיבה`, או `ללא גישה`.

| **תחום הרשאה**       | **תיאור**                                                                    | **אפשרויות גישה מפורטות** |
| :------------------- | :--------------------------------------------------------------------------- | :------------------------ |
| **הגדרות קמפיין**   | עריכת פרטי קמפיין, ניהול אינטגרציות, הגדרת התראות.                           | קריאה, כתיבה, ללא גישה    |
| **ניהול שגרירים**   | הזמנה, הסרה, הקצאת אנשי קשר, צפייה בפעילות שגרירים.                          | קריאה, כתיבה, ללא גישה    |
| **גישה לנתונים ודוחות** | צפייה בתרומות, ייצוא נתונים, גישה לפרטי אנשי קשר, דוחות.                   | קריאה, כתיבה, ללא גישה    |
| **כלי תקשורת**      | ניהול תבניות הודעות, תסריטי שיחה, הודעות מתוזמנות.                         | קריאה, כתיבה, ללא גישה    |
| **ניהול הרשאות משתמשים** | יצירה והקצאת תפקידים מותאמים אישית, ניהול הרשאות משתמשים אחרים. | קריאה, כתיבה, ללא גישה    |

## יצירה והקצאה של תפקידים מותאמים אישית

המתגים הבינאריים הקודמים להרשאות 'מנהל מערכת' ו'מנהל צוות' הוחלפו במטריצת הרשאות מקיפה בתוך ממשק ניהול התפקידים המותאמים אישית החדש.

כדי ליצור תפקידים מותאמים אישית חדשים ולהקצות אותם למשתמשים:

1.  עבור ל**ממשק ניהול**.
2.  נווט אל **הגדרות** > **תפקידים מותאמים אישית** כדי להגדיר תפקידים חדשים ולקבוע את הרשאותיהם המפורטות. לשלבים מפורטים, ראה [יצירה וניהול תפקידים מותאמים אישית](/articles/campaign-management/creating-and-managing-custom-roles).
3.  לאחר הגדרת התפקידים, עבור ל**אנשי קשר** > **ניהול שגרירים**.
4.  לחץ על המשתמש שאת הרשאותיו ברצונך לערוך.
5.  הקצה את התפקידים המותאמים אישית הרצויים מהאפשרויות הזמינות. לשלבים מפורטים, ראה [הקצאת תפקידים מותאמים אישית למשתמשים](/articles/campaign-management/assigning-custom-roles-to-users).
6.  שמור את השינויים.

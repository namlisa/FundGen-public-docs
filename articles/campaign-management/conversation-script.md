# Call Scripts

A Call Script serves as a fixed "conversation skeleton," which can be customized in real-time according to the contact person and the situation.

## Accessing the Admin Interface

- Access the FundGen admin interface via computer or mobile device at the following link: [https://admin.givingl.ink/dashboard](https://admin.givingl.ink/dashboard)
- Log in using a user with appropriate permissions (**Admin Permission**).

## Creating a Call Script

1. In the main toolbar, navigate to the **Messages** category and then select **Call Scripts**.
2. Click the **+** button to add a new script.
3. **Define the script details:**
  - **Script Name** (e.g., "Initial Contact – Ambassador Recruitment").
  - **Script Purpose** (optional but recommended field).

4. **Add steps to the script**, according to the desired conversation structure, such as:
  - Opening message
  - Follow-up message
  - Closing message / Call to action

**Each step in the script may include:**

- Free text
- Emojis

# תסריטי שיחה

תסריט שיחה משמש כ״שלד שיחה״ קבוע, אותו ניתן להתאים אישית בזמן אמת בהתאם לאיש הקשר ולסיטואציה.

## כניסה לממשק הניהול

1. יש להיכנס לממשק הניהול של FundGen באמצעות מחשב או מכשיר נייד בקישור הבא: [https://admin.givingl.ink/dashboard](https://admin.givingl.ink/dashboard)
2. יש להתחבר באמצעות משתמש בעל הרשאות מתאימות (הרשאת מנהל).

## יצירת תסריט שיחה

1. בסרגל הכלים הראשי, יש לנווט לקטגוריית **הודעות** ולאחר מכן לבחור ב־**תסריטי שיחה**.
2. יש ללחוץ על כפתור **+** להוספת תסריט חדש.
3. יש להגדיר את פרטי התסריט:
  - **שם התסריט** (לדוגמה: „פנייה ראשונית – גיוס שגריר”)
  - **מטרת התסריט** (שדה אופציונלי אך מומלץ)

4. יש להוסיף שלבים לתסריט, בהתאם למבנה השיחה הרצוי, כגון:
  - הודעת פתיחה
  - הודעת המשך
  - הודעת סיום / הנעה לפעולה

כל שלב בתסריט יכול לכלול:

- טקסט חופשי
- אימוג’ים

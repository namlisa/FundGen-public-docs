# Conversation Script

The script serves as a fixed "conversation skeleton," which can be customized in real-time according to the contact person and the situation.

## **Accessing the Admin Interface**

- Access the system's admin interface via computer or mobile device at the following link: [https://admin.givingl.ink/dashboard](https://www.google.com/url?sa=E&source=gmail&q=https://admin.givingl.ink/dashboard)
- Log in using a user with appropriate permissions (**Admin Permission**).

## 

## **Creating a Conversation Script**

1. In the main toolbar, enter the **Messages** category and then select **Conversation Scripts**.
2. Click the **+** button to add a new script.
3. **Define the script details:**
  - **Script Name** (e.g., "Initial Contact – Ambassador Recruitment").
  - **Script Purpose** (optional but recommended field).

4. **Add steps to the script**, according to the desired conversation structure, such as:
  - Opening message
  - Follow-up message
  - Closing message / Call to action

**Each step in the script may include:**

- Free text
- Emojis

# תסריט שיחה

לפני שמתחילים: ניתן לצפות במדריך מצולם.

התסריט משמש כ״שלד שיחה״ קבוע, אותו ניתן להתאים אישית בזמן אמת בהתאם לאיש הקשר ולסיטואציה.

## כניסה לממשק הניהול

1. יש להיכנס לממשק הניהול של המערכת באמצעות מחשב או מכשיר נייד בקישור הבא: [https://admin.givingl.ink/dashboard](https://admin.givingl.ink/dashboard)
2. יש להתחבר באמצעות משתמש בעל הרשאות מתאימות (הרשאת מנהל).

## החלפת שפת המערכת לעברית

ברירת המחדל של שפת המערכת היא אנגלית.  
על מנת להציג את הממשק בעברית, יש לבצע את הפעולות הבאות:

1. בסרגל הכלים העליון, יש ללחוץ על הכפתור **CHANGE LANGUAGE**
2. מתוך רשימת השפות, לבחור באפשרות **Hebrew**
3. הממשק יתעדכן אוטומטית לשפה העברית

## יצירת תסריט שיחה

1. בסרגל הכלים הראשי, יש להיכנס לקטגוריית **הודעות** ולאחר מכן לבחור ב־**תסריטי שיחה**
2. יש ללחוץ על כפתור **+** להוספת תסריט חדש
3. יש להגדיר את פרטי התסריט:
  - **שם התסריט** (לדוגמה: „פנייה ראשונית – גיוס שגריר”)
  - **מטרת התסריט** (שדה אופציונלי אך מומלץ)

4. יש להוסיף שלבים לתסריט, בהתאם למבנה השיחה הרצוי, כגון:
  - הודעת פתיחה
  - הודעת המשך
  - הודעת סיום / הנעה לפעולה

כל שלב בתסריט יכול לכלול:

- טקסט חופשי
- אימוג’ים

# Glossary

## **Core Terms**

| **Term**             | **Definition**                                                                           |
|----------------------|------------------------------------------------------------------------------------------|
| **Ambassador**       | A fundraiser who helps solicit money for a campaign.                                   |
| **Donation Manager** | A user role with permission to view all donations, edit donor details, and initiate refunds. |
| **Potential Donor**  | A contact on your fundraising list.                                                      |
| **Campaign**         | A fundraising initiative with a goal and a timeframe.                                  |
| **Activity**         | Any interaction you log with a contact (call, message, etc.).                            |

## **Potential Donor Status Categories**

| **Status**      | **Meaning**                                               |
|-----------------|-----------------------------------------------------------|
| **New**         | A new contact, not yet contacted.                         |
| **In Progress** | Contacted, awaiting a response, or promised to donate.    |
| **Done**        | Achieved a final outcome (donated, declined, or archived). |

## **Potential Donor Statuses**

| **Status**                   | **Meaning**                                             |
|------------------------------|---------------------------------------------------------|
| **Contacted**                | You made contact with the potential donor.              |
| **No reply to contact**      | The potential donor did not answer the call or message. |
| **Promised to donate**       | The potential donor committed to making a donation.     |
| **Donated**                  | The potential donor made a donation.                    |
| **Not interested**           | The potential donor declined to donate.                 |
| **Wrong contact details**    | The potential donor's contact details are incorrect.    |
| **Contact later**            | The potential donor should be contacted at a later time. |
| **Maybe donate in the future** | The potential donor expressed interest but not commitment yet. |
| **Donated by other**         | The potential donor already donated through another ambassador. |
| **Thanked**                  | A thank you message has been sent to the donor.         |

## **Gamification**

| **Term** | **Definition**                                  |
|----------|-------------------------------------------------|
| **XP**   | Experience points earned for activities.        |
| **Level**| Your progress rank based on XP.                 |
| **Badge**| An achievement earned for milestones.           |
| **Tip**  | An insight shared with the team after a donation. |

## **Platform Terms**

| **Term**                 | **Definition**                                          |
|--------------------------|---------------------------------------------------------|
| **Personal Link**        | Your unique donation page URL.                          |
| **Potential Donors**     | Contacts uploaded to the database by the campaign manager. |
| **Tap to Pay**           | Contactless card payment via phone.                     |
| **Deep Dive Research**   | Background search on a donor using AI.                  |

# מילון מונחים

## **מונחי יסוד**

| **מונח**          | **הגדרה**                                                                             |
|-------------------|---------------------------------------------------------------------------------------|
| **שגריר**         | מגייס כספים שעוזר לגייס כסף לקמפיין.                                                 |
| **מנהל תרומות**   | תפקיד משתמש עם הרשאה לצפות בכל התרומות, לערוך פרטי תורמים וליזום החזרים כספיים.      |
| **תורם פוטנציאלי**| איש קשר ברשימת הגיוס שלך.                                                            |
| **קמפיין**        | יוזמת גיוס כספים עם יעד ומסגרת זמן.                                                   |
| **פעילות**        | כל אינטראקציה שאתה מתעד עם איש קשר (שיחה, הודעה וכו').                               |

## **קטגוריות סטטוסים של תורמים פוטנציאליים**

| **סטטוס**    | **משמעות**                                      |
|--------------|-------------------------------------------------|
| **חדש**      | איש קשר חדש, עוד לא יצרת קשר.                   |
| **בתהליך**   | יצרת קשר, ממתין לתשובה או הבטיח לתרום.          |
| **בוצע**     | הגעת לתוצאה סופית (תרם, סירב, או הועבר לארכיון). |

## **סטטוסי תורמים פוטנציאליים**

| **סטטוס**             | **משמעות**                                            |
|-----------------------|-------------------------------------------------------|
| **יצרתי קשר**         | יצרת קשר עם התורם הפוטנציאלי.                         |
| **לא ענה לפנייה**     | התורם הפוטנציאלי לא ענה לשיחה או להודעה.             |
| **התחייב לתרום**     | התורם הפוטנציאלי הבטיח לבצע תרומה.                   |
| **תרם**               | התורם הפוטנציאלי ביצע תרומה.                         |
| **לא מעוניין**       | התורם הפוטנציאלי סירב לתרום.                         |
| **פרטי קשר שגויים** | פרטי הקשר של התורם הפוטנציאלי אינם נכונים.           |
| **ליצור קשר מאוחר יותר**| יש ליצור קשר עם התורם הפוטנציאלי במועד מאוחר יותר. |
| **אולי יתרום בעתיד**  | התורם הפוטנציאלי הביע עניין אך לא התחייב כרגע.      |
| **נתרם על ידי אחר** | התורם הפוטנציאלי תרם כבר דרך שגריר אחר.             |
| **נשלחה תודה**       | נשלחה הודעת תודה לתורם.                             |

## **גיימיפיקציה**

| **מונח**  | **הגדרה**                                      |
|-----------|------------------------------------------------|
| **XP**    | נקודות ניסיון שנצברות עבור פעילויות.           |
| **רמה**   | דרגת ההתקדמות שלך על בסיס XP.                 |
| **תג**    | הישג שנצבר עבור אבני דרך.                      |
| **טיפ**   | תובנה שמשותפת עם הצוות אחרי תרומה.            |

## **מונחי פלטפורמה**

| **מונח**             | **הגדרה**                                         |
|----------------------|---------------------------------------------------|
| **לינק אישי**        | כתובת דף התרומה האישי שלך.                       |
| **תורמים פוטנציאליים**| אנשי קשר שמנהל הקמפיין העלה לבסיס הנתונים.     |
| **הקש לשלם**        | תשלום כרטיס ללא מגע דרך הטלפון.                   |
| **מחקר מעמיק**       | חיפוש רקע על תורם באמצעות AI.                   |

# מילון מונחים

## **מונחי יסוד**

| **מונח** | **הגדרה** |
|---|---|
| **שגריר** | מגייס כספים שעוזר לגייס כסף לקמפיין |
| **תורם פוטנציאלי** | איש קשר ברשימת הגיוס שלך |
| **קמפיין** | יוזמת גיוס כספים עם יעד ומסגרת זמן |
| **פעילות** | כל אינטראקציה שאתה מתעד עם איש קשר (שיחה, הודעה וכו') |

## **קטגוריות סטטוסים של תורמים פוטנציאליים**

| **סטטוס** | **משמעות** |
|---|---|
| **חדש** | איש קשר חדש, עוד לא יצרת קשר |
| **בתהליך** | יצרת קשר, ממתין לתשובה או הבטיח לתרום |
| **בוצע** | הגעת לתוצאה סופית (תרם, סירב, או הועבר לארכיון) |

## **סטטוסי תורמים פוטנציאליים**

| **סטטוס** | **משמעות** |
|---|---|
| **יצרתי קשר** | יצרת קשר עם התורם הפוטנציאלי |
| **לא ענה לפנייה** | התורם הפוטנציאלי לא ענה לשיחה או להודעה |
| **התחייב לתרום** | התורם הפוטנציאלי הבטיח לבצע תרומה |
| **תרם** | התורם הפוטנציאלי ביצע תרומה |
| **לא מעוניין** | התורם הפוטנציאלי סירב לתרום |
| **פרטי קשר שגויים** | פרטי הקשר של התורם הפוטנציאלי אינם נכונים |
| **ליצור קשר מאוחר יותר** | יש ליצור קשר עם התורם הפוטנציאלי במועד מאוחר יותר |
| **אולי יתרום בעתיד** | התורם הפוטנציאלי הביע עניין אך לא התחייב כרגע |
| **נתרם על ידי אחר** | התורם הפוטנציאלי תרם כבר דרך שגריר אחר |
| **נשלחה תודה** | נשלחה הודעת תודה לתורם |

## **גיימיפיקציה**

| **מונח** | **הגדרה** |
|---|---|
| **XP** | נקודות ניסיון שנצברות עבור פעילויות |
| **רמה** | דרגת ההתקדמות שלך על בסיס XP |
| **תג** | הישג שנצבר עבור אבני דרך |
| **טיפ** | תובנה שמשותפת עם הצוות אחרי תרומה |

## **מונחי פלטפורמה**

| **מונח** | **הגדרה** |
|---|---|
| **לינק אישי** | כתובת דף התרומה האישי שלך |
| **דף תרומה מותאם אישית (חדש!)** | דף תרומה הניתן להתאמה אישית מלאה עם מיתוג ופרטים ספציפיים לקמפיין. |
| **תמיכה בריבוי מטבעות (חדש!)** | היכולת לקבל תרומות במספר מטבעות שונים (לדוגמה: USD, EUR, GBP, ILS). |
| **יעד קמפיין הושג (התראה חדשה!)** | התראה אוטומטית הנשלחת כאשר סך היעד הכולל של הקמפיין הושג. |
| **השבת שגריר (חדש!)** | פעולה של מנהל קמפיין המונעת משגריר גישה לאפליקציה או ליצירת קישורי תרומה חדשים, ללא מחיקת הנתונים ההיסטוריים שלו. |
| **שיוך אנשי קשר בכמות גדולה (חדש!)** | היכולת של מנהלי קמפיין לשייך קבוצה של אנשי קשר למספר שגרירים או צוותים בו-זמנית. |
| **שיוך אנשי קשר לפי קריטריונים (חדש!)** | היכולת לשייך אנשי קשר לשגרירים או צוותים על בסיס מאפיינים ספציפיים (לדוגמה: תגיות או מקור). |
| **תורמים פוטנציאליים** | אנשי קשר שמנהל הקמפיין העלה לבסיס הנתונים |
| **הקש לשלם** | תשלום כרטיס ללא מגע דרך הטלפון |
| **מחקר מעמיק** | חיפוש רקע על תורם באמצעות AI |

# הורדת האפליקציה

# 

## **קישור הורדה**

**אייפון (iOS) + אנדרואיד**

- קישור להורדה - https://ambassador.app.link/wss 

## **שלבי התקנה**

## **אייפון**

1. פתח את App Store
2. חפש "Ambassador App"
3. לחץ **קבל** להורדה
4. פתח את האפליקציה לאחר ההתקנה

## **אנדרואיד**

1. פתח את Google Play Store
2. חפש "Ambassador App"
3. לחץ **התקן**
4. פתח את האפליקציה לאחר ההתקנה

## **לאחר ההתקנה**

לאחר ההתקנה, פתח את האפליקציה, והתחבר בעזרת מספר הפלאפון שלך.

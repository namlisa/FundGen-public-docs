# סקירת אינטגרציית וואטסאפ

אפליקציית FundGen Ambassador מציעה אינטגרציה עמוקה עם וואטסאפ, המאפשרת למנהלי קמפיינים ולשגרירים לייעל את התקשורת. כעת, מנהלים יכולים להגדיר ולנהל מספר פרופילי בוט לוואטסאפ, כל אחד עם הגדרות ייחודיות ומשויך לקבוצת קמפיין ספציפית.

## **יכולות האינטגרציה**

| **תכונה** | **תיאור** |
|---|---|
| הודעות ישירות | שליחה דרך וואטסאפ |
| יצירת קבוצות | תיאום צוות |
| שיתוף קישורים | קישורי תרומה עקיבים |
| התראות אוטומטיות (דרך בוטים) | התראות מותאמות אישית בזמן אמת, מוגדרות לכל פרופיל בוט בנפרד |
| **ניהול פרופילי בוט מרובים** | הקצאת בוטים שונים לקבוצות קמפיין שונות עם מספרי וואטסאפ והגדרות אישיות |

## **איך זה עובד**

אפליקציית FundGen Ambassador משתלבת עם וואטסאפ על ידי:

1.  פתיחת וואטסאפ עם הודעות מלאות מראש (לשגרירים)
2.  מעקב אחר הודעות שנשלחו כפעילויות
3.  אספקת תבניות הודעות לעקביות
4.  יצירת קבוצות פרוגרמטית
5.  **הפעלת פרופילי בוט מרובים המחוברים לקבוצות וואטסאפ ספציפיות ושולחים הודעות אוטומטיות מותאמות אישית.**

## **זרימת הודעות**

```
`אפליקציית FundGen Ambassador → מילוי הודעה מראש → פתיחת וואטסאפ → המשתמש שולח → רישום פעילות`
`אפליקציית FundGen Ambassador → הגדרת בוט (בממשק ניהול) → שליחת התראות אוטומטיות לקבוצה (דרך מספר וואטסאפ של המנהל המקושר לבוט)`
```

## **דרישות**

| **דרישה** | **פרטים** |
|---|---|
| אפליקציית וואטסאפ | מותקנת במכשיר |
| מספר טלפון | רשום בוואטסאפ |
| מספרי אנשי קשר | מספרים סלולריים תקינים |
| אינטרנט | חיבור פעיל |
| **פרופילי בוט לוואטסאפ** | דורש חיבור של מספר וואטסאפ לכל פרופיל בוט שנוצר |

## **תכונות זמינות**

## **לשגרירים**

-   שליחת הודעות מותאמות אישית
-   שימוש בתבניות הודעות
-   מעקב אחר הודעות שנשלחו
-   שיתוף קישורי תרומה

## **למנהלים**

-   **יצירה וניהול של פרופילי בוט מרובים לוואטסאפ.**
-   יצירת קבוצות צוות
-   **הגדרת התראות תרומה והודעות מתוזמנות נוספות עם ניתוב לקבוצות ספציפיות, באופן פרטני עבור כל פרופיל בוט.**
-   הגדרת תבניות הודעות
-   ניטור פעילות הודעות ובוטים

## **שיקולי פרטיות**

-   הודעות נשלחות דרך וואטסאפ של המשתמש (לשגרירים) או דרך מספר הוואטסאפ המחובר לפרופיל הבוט (לבוטים).
-   אין תוכן הודעות נשמר על ידי אפליקציית FundGen Ambassador (פרט לתבניות).
-   פעילות נרשמת (לא תוכן מלא של הודעות שנשלחו באופן ידני).
-   המשתמש שולט בשיתוף.

## **מגבלות**

-   לא ניתן לשלוח הודעות אוטומטיות (בצד השגריר, אלא אם מדובר בבוט). בוטים שולחים הודעות אוטומטיות כפי שהוגדר.
-   דורש פעולת משתמש לשליחה (לשגרירים).
-   איש הקשר חייב להיות בוואטסאפ.
-   יצירת קבוצה דורשת הרשאות.
-   **כל פרופיל בוט לוואטסאפ דורש חיבור למספר וואטסאפ ייעודי.**

# WhatsApp Integration Overview

The FundGen Ambassador App offers deep integration with WhatsApp, allowing campaign managers and ambassadors to streamline communication. Now, managers can configure and manage multiple WhatsApp Bot Profiles, each with unique settings and assigned to a specific campaign group.

## **Integration Capabilities**

| **Feature** | **Description** |
|---|---|
| Direct Messages | Sending via WhatsApp |
| Group Creation | Team Coordination |
| Link Sharing | Consistent Donation Links |
| Automated Alerts (via Bots) | Customized real-time alerts, configured individually for each bot profile |
| **Multiple Bot Profiles Management** | Assigning different bots to different campaign groups with unique WhatsApp numbers and personalized settings |

## **How it Works**

The FundGen Ambassador App integrates with WhatsApp by:

1.  Opening WhatsApp with pre-filled messages (for ambassadors)
2.  Tracking sent messages as activities
3.  Providing message templates for consistency
4.  Programmatically creating groups
5.  **Operating multiple bot profiles connected to specific WhatsApp groups, sending tailored automated messages.**

## **Message Flow**

```
`FundGen Ambassador App → Pre-fill Message → Open WhatsApp → User Sends → Activity Logged`
`FundGen Ambassador App → Bot Configuration (Admin Interface) → Automated Alerts to Group (via manager's WhatsApp number linked to bot)`
```

## **Requirements**

| **Requirement** | **Details** |
|---|---|
| WhatsApp App | Installed on device |
| Phone Number | Registered with WhatsApp |
| Contact Numbers | Valid mobile numbers |
| Internet | Active connection |
| **WhatsApp Bot Profiles** | Each bot profile requires a dedicated WhatsApp number connection |

## **Available Features**

## **For Ambassadors**

-   Sending personalized messages
-   Using message templates
-   Tracking sent messages
-   Sharing donation links

## **For Managers**

-   **Creating and managing multiple WhatsApp Bot Profiles.**
-   Creating team groups
-   **Configuring donation and milestone alerts individually for each bot profile and group.**
-   Setting up message templates
-   Monitoring message and bot activity

## **Privacy Considerations**

-   Messages are sent via the user's WhatsApp (for ambassadors) or via the WhatsApp number connected to the bot profile (for bots).
-   No message content is stored by the FundGen Ambassador App (except for templates).
-   Activity is logged (not full content of manually sent messages).
-   The user controls sharing.

## **Limitations**

-   Automated messages cannot be sent (by the ambassador, unless it's a bot). Bots send automated messages as configured.
-   Requires user action to send (for ambassadors).
-   Contact must be on WhatsApp.
-   Group creation requires permissions.
-   **Each WhatsApp Bot Profile requires a connection to a dedicated WhatsApp number.**

# סקירת אינטגרציית וואטסאפ

## **יכולות האינטגרציה**

| **תכונה** | **תיאור** |
|---|
| הודעות ישירות | שליחה דרך וואטסאפ |
| יצירת קבוצות | תיאום צוות |
| שיתוף קישורים | קישורי תרומה עקיבים |
| התראות | התראות בזמן אמת |

## **איך זה עובד**

אפליקציית FundGen Ambassador משתלבת עם וואטסאפ על ידי:

1.  פתיחת וואטסאפ עם הודעות מלאות מראש
2.  מעקב אחר הודעות שנשלחו כפעילויות
3.  אספקת תבניות הודעות לעקביות
4.  יצירת קבוצות פרוגרמטית

## **זרימת הודעות**

```
`אפליקציית FundGen Ambassador → מילוי הודעה מראש → פתיחת וואטסאפ → המשתמש שולח → רישום פעילות`
```

## **דרישות**

| **דרישה** | **פרטים** |
|---|
| אפליקציית וואטסאפ | מותקנת במכשיר |
| מספר טלפון | רשום בוואטסאפ |
| מספרי אנשי קשר | מספרים סלולריים תקינים |
| אינטרנט | חיבור פעיל |

## **תכונות זמינות**

## **לשגרירים**

-   שליחת הודעות מותאמות אישית
-   שימוש בתבניות הודעות
-   מעקב אחר הודעות שנשלחו
-   שיתוף קישורי תרומה

## **למנהלים**

-   יצירת קבוצות צוות
-   הגדרת התראות תרומה והודעות מתוזמנות נוספות עם ניתוב לקבוצות ספציפיות
-   הגדרת תבניות הודעות
-   ניטור פעילות הודעות

## **שיקולי פרטיות**

-   הודעות נשלחות דרך וואטסאפ של המשתמש
-   אין תוכן הודעות נשמר על ידי אפליקציית FundGen Ambassador
-   פעילות נרשמת (לא תוכן)
-   המשתמש שולט בשיתוף

## **מגבלות**

-   לא ניתן לשלוח הודעות אוטומטיות
-   דורש פעולת משתמש לשליחה
-   איש הקשר חייב להיות בוואטסאפ
-   יצירת קבוצה דורשת הרשאות

-   סקירה: תכונות וואטסאפ
-   יצירת קבוצת וואטסאפ

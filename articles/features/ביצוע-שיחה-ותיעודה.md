# ביצוע שיחה ותיעודה

# 

## **שלבי התקשרות**

1. פתח את פרופיל איש הקשר
2. לחץ על כפתור **📞 שיחה**
3. החייגן של הטלפון נפתח עם המספר
4. בצע את השיחה
5. חזור לאפליקציית השגרירים לאחר השיחה
6. תעד את התוצאה (הנחיה מופיעה אוטומטית)

## **תיעוד תוצאת השיחה**

לאחר השיחה, בחר מה קרה:

| **תוצאה** | **מתי להשתמש** |
|---|---|
| **יצרתי קשר** | דיברת איתם |
| **לא ענה** | לא ענו |
| **השארתי הודעה** | השארת הודעה קולית |
| **הבטיח לתרום** | התחייב לתת |
| **תרם** | ביצע תרומה בשיחה |
| **לא מעוניין** | סירב |
| **ליצור קשר מאוחר יותר** | ביקש שתתקשר חזרה |

## **הוספת הערות**

1. לאחר בחירת התוצאה, אפשר להוסיף הערות
2. כתוב פרטים רלוונטיים:
  - "יתרום 500$ בשבוע הבא"
  - "ביקש להתקשר אחרי 18:00"
  - "מעוניין אבל מחכה למשכורת"

3. לחץ **הוספת הערה**

## **הגדרת מעקב**

אם בחרת "ליצור קשר מאוחר יותר":

1. בחר תאריך ושעה להתקשרות חזרה
2. תקבל התראת תזכורת
3. איש הקשר נשאר בעמודת "בתהליך"

# הגדרת תזכורת מעקב

# 

## **מתי להגדיר תזכורות**

- איש קשר ביקש שתתקשר בזמן ספציפי
- רוצה לעקוב אחרי שליחת מידע
- איש קשר אמר "אולי מאוחר יותר"
- צריך לבדוק על תרומה שהובטחה

## **שלבי הגדרת תזכורת**

## **דרך 1: במהלך תיעוד פעילות**

1. לחץ על אייקון +
2. בחר **יש ליצור קשר מאוחר יותר**
3. בחר תאריך ושעה
4. לחץ **הזכר לי**

## **מה קורה**

1. התזכורת נשמרת לאיש הקשר
2. בזמן המתוזמן, תקבל:
  - התראת push בטלפון
  - התראה באפליקציה

3. לחץ על ההתראה לפתיחת איש הקשר ישירות

## **התראת תזכורת**

```
`⏰ תזכורת  התקשר לדוד כהן  "הוא ביקש להתקשר אחרי 18:00"  לחץ לפתיחת איש הקשר`
```

## **ניהול תזכורות**

## **צפייה בכל התזכורות**

1. עבור ללשונית **התראות** - אייקון של פעמון
2. צפה בכל ההתראות

## **ביטול תזכורת**

1. פתח את איש הקשר
2. מצא את התזכורת
3. לחץ **מחק**

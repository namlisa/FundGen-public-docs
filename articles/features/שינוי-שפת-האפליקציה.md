# שינוי שפת האפליקציה

# 

## **שינוי השפה**

1. עבור ל**פרופיל** > **הגדרות**
2. מצא **שפה**
3. לחץ לבחירה
4. בחר את השפה שלך
5. האפליקציה מופעלת מחדש עם שפה חדשה

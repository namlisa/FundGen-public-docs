# יצירת קבוצת וואטסאפ

# 

## **יצירת קבוצה מאפליקציית השגרירים**

1. עבור ל**ממשק ניהול** > **וואטסאפ**
2. לחץ **צור קבוצה**
3. הזן שם קבוצה
4. בחר חברים (שגרירים)
5. לחץ **צור**
6. קבוצה נוצרה בוואטסאפ

## **מה קורה**

כשיוצרים קבוצה:

1. וואטסאפ נפתח במכשיר שלך
2. הקבוצה מוגדרת מראש
3. שגרירים נבחרים מוזמנים
4. אתה נעשה מנהל

## **הגדרות קבוצה**

הגדר באפליקציית השגרירים:

- **שם** - שם תצוגה של הקבוצה
- **תיאור** - מטרת הקבוצה
- **חברים** - מי כלול
- **הרשאות** - מי יכול לפרסם

## **שימוש בקבוצות להתראות**

לשליחת התראות תרומה לקבוצה:

1. צור קבוצת התראות ייעודית
2. עבור ל**הודעות מתוזמנות** > **וואטסאפ**
3. בחר קבוצה כנמען
4. הפעל סוגי התראות
5. שמור

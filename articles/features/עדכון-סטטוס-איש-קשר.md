# עדכון סטטוס איש קשר

# 

## **מתי לעדכן סטטוס**

- איש קשר שאמר "לא" עכשיו רוצה לתרום
- איש קשר שהבטיח עבר ל"תרם"
- מספר שגוי היה בעצם נכון
- צריך לחדש מעורבות עם איש קשר בארכיון

## **שלבי עדכון**

1. פתח את פרופיל איש הקשר
2. לחץ **שנה סטטוס** או על מחוון הסטטוס
3. בחר את הסטטוס החדש
4. תעד פעילות שמסבירה את השינוי (אופציונלי אבל מומלץ)
5. לחץ **שמור**

## **מעברי סטטוס**

| **מ** | **ל** | **מתי** |
|---|---|---|
| חדש | בתהליך | יצרת קשר ראשון |
| בתהליך | בוצע | קיבלת תרומה או תשובה סופית |
| בוצע | בתהליך | מחדש מעורבות עם איש קשר סגור |
| בוצע | חדש | איפוס לקמפיין חדש |

## **שינוי תוצאות**

לשינוי תוצאה סופית (למשל "לא מעוניין" ל"הבטיח"):

1. פתח פרופיל איש קשר
2. תעד פעילות חדשה עם תוצאה נכונה
3. איש הקשר מתעדכן אוטומטית
4. הוסף הערה שמסבירה את השינוי

## **שחזור אנשי קשר מהארכיון**

1. עבור ל**לידים** > עמודת **בוצע**
2. מצא את איש הקשר בארכיון
3. פתח את הפרופיל שלו
4. לחץ **שחזר** או תעד פעילות חיובית חדשה
5. איש הקשר חוזר לסטטוס פעיל

## **הערה**

שינוי סטטוס לא מוחק היסטוריית פעילות. כל הפעילויות הקודמות נשארות בציר הזמן לעיון.

---

**מאמרים קשורים:**

- העברת אנשי קשר בין עמודות
- סימון איש קשר כלא מעוניין
